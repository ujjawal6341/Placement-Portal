Project Details

Project Title: Placement Portal(Naukri Shala)
Problem Statement:
The traditional placement process in many institutions involves manual coordination between students, companies, and administrators. This often
leads to inefficiencies such as delayed communication, difficulty in tracking applications, and lack of centralized information.
The objective of this project is to design and develop a web based placement management system that allows students to apply for job
opportunities, companies to post job drives, and administrators to manage the entire recruitment process efficiently.
Approach:
The application was developed using the Flask web framework with a modular architecture. The system includes three primary roles:
● Students who can register, view job drives, and apply for positions.
● Companies who can register, create recruitment drives, and review student applications.
● Administrators who manage approvals, monitor companies, and oversee the placement process.
The backend is implemented using Flask and SQLAlchemy, while the frontend uses HTML, Bootstrap, and Jinja2 templates for dynamic rendering.

AI / LLM Declaration

I used ChatGPT (GPT-5) to assist with:
● Improving the structure of Flask routes and modular project architecture
● Debugging SQLAlchemy database errors
● Writing parts of documentation and improving template structure
The approximate usage of AI assistance is 10–15%, mainly for guidance and debugging suggestions.
All major components including project logic, debugging, database design, and final integration were implemented and tested
manually.
